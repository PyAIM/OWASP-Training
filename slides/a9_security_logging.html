A9: Security Logging and Monitoring Failures

Detecting and responding to security breaches

What are Security Logging Failures?

Security logging and monitoring failures occur when applications don't properly log, detect, escalate, or respond to security events. Without adequate logging and monitoring, breaches can go undetected for extended periods, allowing attackers to maintain persistence and cause significant damage.
Missing Logs
No logging for critical security events
Unclear Logs
Logs without sufficient context or details

Impact of Insufficient Logging

  • Attackers can remain undetected for extended periods
  • Security breaches may only be discovered after significant damage
  • Forensic investigations become difficult or impossible
  • Compliance violations due to inability to track user actions
CTF Challenge
Objective: Find evidence of a security breach in the application logs.
Hint: Use Burp Suite to perform multiple failed login attempts, then check if they are properly logged.
