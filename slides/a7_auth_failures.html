A7: Authentication Failures

Exploiting weaknesses in authentication mechanisms

What are Authentication Failures?

Authentication failures occur when an application's authentication mechanisms are improperly implemented, allowing attackers to compromise passwords, keys, session tokens, or exploit implementation flaws to assume other users' identities.
Credential Stuffing
Using stolen credentials from one site to access other sites
Brute Force
Systematically trying all possible passwords

Session Hijacking

Session hijacking occurs when an attacker steals or manipulates a user's session token to gain unauthorized access to their account. This can happen through various methods including XSS attacks, network sniffing, or session fixation.
