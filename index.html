Environment Setup

Get your penetration testing environment ready

1. OWASP Juice Shop

docker pull bkimminich/juice-shop
docker run -d -p 3000:3000 juice-shop

Set up the vulnerable web application for testing

2. FoxyProxy

Install FoxyProxy extension in Firefox

Configure proxy: 127.0.0.1:8080

Route traffic through Burp Suite proxy

3. Burp Suite

Download Burp Suite Community

Configure proxy listener on port 8080

Intercept and analyze web traffic
